## Live Server
受到 [Visual Studio Live Server](https://github.com/ritwickdey/vscode-live-server) 启发打造

一个简单易用的 Web 服务器，提供方便的网页预览功能

### 它能干些啥
直接在当前目录启动一个 Web 服务器，展示当前目录下的网站内容

以下为一个示例，展示了把 LiveServer 程序添加到环境变量后预览 Vue 项目构建产物的过程
